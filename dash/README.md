# Vision

Handheld gadgets to make it easy to manage Descent tokens during play

## Hero dashboard

Here is a vision of the gadget, created using AI.

![[readme-images/deevidai_generated.jpg]]

The idea is to not have to use tokens at all, but keep all current status effects, chosen actions, coin, health, fatigue, and potions as counters in the gadget.

### Functionality

At its very least, the dashboard serves to record everything during the game without having to use any cardboard tokens.

At best, it connects to a server and transmits hero actions to it.
